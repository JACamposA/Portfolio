¿Quién soy?

Mi nombre es José A. Campos Arbulú, soy un Desarrollador y Maquetador Web, con conocimientos de Javascript, HTML5, CSS, JQuery y Bootstrap. Estoy en constante aprendizaje y me apasionan los desafíos. Actualmente estoy aprendiendo a programar con React Native y pronto lo sumaré a la lista de conocimientos que acabas de leer

El maquetado web es la construcción de una página o aplicación desde cero, basado en un diseño propio o pedido por el cliente, se maneja tradicionalmente con HTML5 y CSS pero nuevas tecnologías como React y Angular están revolucionando el maquetado.

Es el desarrollo de sitios web y aplicaciones, incluyendo sus funcionalidades con javascript, animaciones y otras especificaciones del cliente, Jquery es una de las librerías de Javascript que manejo para este propósito.

El día de hoy la gran mayoría del tráfico por internet es realizado en el celular, el mobile first optimiza las páginas web para que se naveguen rápida y fácilmente en el teléfono.
